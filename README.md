Fantasy Premier League AI Agent Suite
Overview

This project provides a suite of AI agents to assist Fantasy Premier League (FPL) managers in planning, analyzing, and optimizing their teams. Inspired by technical blogging agents, this suite adapts them to the FPL domain, allowing for team strategy generation, transfer recommendations, detailed reports, and optional social insights.

Features

Team Planning: Generates optimal starting lineups and transfer suggestions based on player stats, fixtures, and budget constraints.

Team Analysis: Produces in-depth reports including player performance, risk assessment, and reasoning for each recommended transfer.

Interactive Feedback: Allows iterative refinement of strategies and reports based on user feedback.

Social Insights: Optional generation of FPL insights suitable for sharing on social media platforms.

Export Capability: Save detailed reports as Markdown files for offline review.
